Patricia's Portfolio

Welcome to my portfolio! Here are some of my Web Development in Client-Side Environment projects.

Projects

Date Management

An interactive countdown timer that allows users to select a date and see the remaining time in months, days, hours, minutes and seconds. The colors change based on the urgency of the event.

Exploding Cards

Inspired by the Exploding Kittens board game, this interactive app allows players to draw from a deck of various card types until it's empty. Users can also reset the game to play again.

SplitWise Clone

A web application for managing shared expenses among friends, allowing users to add expenses, view balances, and calculate debts in a simple and effective manner.

Customizer

This project allows users to customize a virtual T-shirt by adding dynamic text, moving it with sliders, changing the shirt color, and dragging logos to place on the shirt. It also updates the design in real-time, adjusting the layout and logo position based on user interactions.

Sticky Notes

This project is a sticky notes application that uses IndexedDB for persistent storage. It allows users to create, move, edit, and delete notes, with all changes automatically saved to the database.

Gas Stations

This app allows users to search for gas stations based on province, municipality, and product. It displays essential information such as the station's address, schedule, and available products. Additionally, users can filter results to show only stations that are currently open.
